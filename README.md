# vlc-pipe
A batch script to watch youtube videos on VLC media player.


## How to use
0. Download the script `vlc-pipe.bat`.
1. Make sure you have VLC Media player and youtube-dl.
2. Make sure VLC and youtube-dl are added to `%PATH%`.
3. Run the script.
